# RScriptServer

## **RScriptServer**

**서버 스크립트에 "Script"로 제공되는 객체로 스크립트가 붙어있는 오브젝트를 Parent로 얻을 수 있다.**

## \*\*\*\*

## **Property**

| **var Parent** |
| :--- |
| **해당 스크립트가 붙어있는 오브젝트** |

| **var Item** |
| :--- |
| **해당 스크립트가 붙어있는 아이템** |

## \*\*\*\*

## **Sample**

```lua
local Object = Script.Parent
local Item = Script.Item
```
