# **RLogger**


로그를 종류별로 출력 
## **함수**

| **Log(string message)** |
| :--- |

일반 로그 메시지 출력 (메시지 내용) 
| **Warning(string message)** |
| :--- |

경고 로그 메시지 출력 (메시지 내용) 
| **Error(string message)** |
| :--- |

에러 로그 메시지 출력 (메시지 내용) 
## **샘플**


```lua
--Hello 메시지를 출력한다.
Logger:Log("Hello")
```
