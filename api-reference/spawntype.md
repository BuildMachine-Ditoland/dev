# SpawnType

스폰 방식을 결정하는 타입이에요   


## **Enums**

| **이름** | **설명** |
| :--- | :--- |
| **UseSpawnGroup** | 스폰그룹을 사용하는 스폰방식이에요. |
| **UseTeamSpawn** | 팀을 사용하는 스폰방식이에요. |
| **UseSpawnPoint** | 스폰포인트를 사용하는 스폰방식이에요. |

샘플

```lua
Game:SetSpawnType(Enum.SpawnType.UseTeamSpawn) --게임의 캐릭터 스폰 방식을 설정해요.
```
